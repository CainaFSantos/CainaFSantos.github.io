Linguagens

  • HTML, CSS e JavaScript

    HTML é usado para estruturar o conteúdo de uma página web. CSS controla o estilo e o layout. JavaScript proporciona interatividade e funcionalidade dinâmica. Essas três tecnologias trabalham em conjunto e são fundamentais para o desenvolvimento web moderno.

  • Bootstrap

    Bootstrap é um popular framework de desenvolvimento web que simplifica a criação de páginas responsivas com componentes pré-projetados. Ele acelera o desenvolvimento e garante uma experiência consistente em diversos dispositivos.

  • Tailwind

    Tailwind CSS é um framework CSS utilitário que simplifica o desenvolvimento web ao oferecer classes pré-definidas para criar rapidamente interfaces de usuário responsivas e personalizáveis, economizando tempo e esforço na escrita de CSS personalizado.

  • Sass

    Sass é uma linguagem de extensão para CSS que oferece recursos avançados, como variáveis e aninhamento, para tornar a escrita e manutenção de estilos web mais eficientes e organizados.

  • Bulma

    Bulma é um framework CSS de código aberto que simplifica o desenvolvimento de designs responsivos e elegantes para sites e aplicativos web, fornecendo classes CSS pré-definidas para criar componentes e estruturas de página de forma eficiente e flexível.

  • ReactJS

    ReactJS é uma biblioteca JavaScript popular para criação de interfaces de usuário interativas e reativas. Sua abordagem baseada em componentes e atualizações eficientes tornam-no uma escolha poderosa para o desenvolvimento ágil e escalável de aplicações web modernas.

  • Vue.js

    Vue.js é um framework JavaScript de código-aberto, focado no desenvolvimento de interfaces de usuário e aplicativos de página única.

  • Next.js

    Next.js é um framework baseado em React, ele oferece recursos como renderização no lado do servidor, roteamento automático e otimização de desempenho, tornando-o uma escolha popular para projetos web complexos.

  • TypeScript

    TypeScript é uma linguagem de programação de código aberto desenvolvida pela Microsoft. É um superconjunto sintático estrito de JavaScript e adiciona tipagem estática opcional à linguagem.

  • Python

    Python é uma linguagem de programação popular de alto nível, conhecida por sua simplicidade e versatilidade, amplamente usada em desenvolvimento web, ciência de dados e automação.

  • PHP

    PHP é uma linguagem de programação amplamente utilizada para desenvolvimento web. Ela é conhecida por sua flexibilidade e capacidade de criar páginas dinâmicas e interativas. Com uma vasta comunidade e uma longa história, o PHP continua sendo uma escolha popular para a criação de aplicativos web.

  • Node JS

    Node.js é um software de código aberto, multiplataforma, baseado no interpretador V8 do Google e que permite a execução de códigos JavaScript fora de um navegador web.

  • MySQL

    O MySQL é um sistema de gerenciamento de banco de dados, que utiliza a linguagem SQL como interface. É atualmente um dos sistemas de gerenciamento de bancos de dados mais populares da Oracle Corporation, com mais de 10 milhões de instalações pelo mundo.
